Чек лист тестирования формы регистрации на сайте:

1.Проверка User Interface:
- наличие плейсхолдеров
- правильность написания названия полей
- правильность расположения полей в соответствии с требованиями 
- проверка видимости полей на темной теме браузера
- проверка видимости текста на темной теме браузера
- наличие всех полей в соответствии с требованиями
- наличие текстового курсора в полях для ввода

2.Проверка регистрации пользователя:
- успешная регистрация, ввести валидные данные 
- неуспешная регистрация, ввести невалидные данные
- неуспешная регистрация, заполнить не все обязательные поля
- проверка письма на почте с подтверждением регистрации
- перегрузить страницу с пройденной регистрацией на проверку не вылетел ли пользователь со страницы, как зарегистрированный пользователь
- ввод уже существующих данных
- заполнить все поля валидными данными и нажать кнопку "Отмена"
- заполнить все поля невалидными данными и нажать кнопку "Отмена"

3.Проверка поля "логин":
- ввести валидное значение
- проверка появления подсказки, при вводе невалидного значения 
- ввод не на латинице
- ввод только цифры
- ввод в логине несколько символов подряд
- ввод логина с заглавными буквами 
- ввод только спецсимволы
- ввод только пробел
- ввод два пробела подряд
- ввод пробел в начале строки
- ввод пробел в середине строки
- ввод пробел в конце строки
- ввод логин больше количества допустимых символов 
- ввод логин меньше количества допустимых символов
- вставить скопированный текст
- перенос строки с помощью tab или enter

4.Проверка поля "email":
- ввести валидное значение
- проверка появления подсказки, при вводе не валидного значения 
- ввод не на латинице
- ввод только спецсимволы
- ввод email только из цифр
- ввод только пробел
- ввод пробел в начале строки
- ввод пробел в середине строки
- ввод пробел в конце строки
- ввод email до "@" больше количества допустимых символов 
- ввод email до "@" меньше количества допустимых символов
- ввод email после "@" больше количества допустимых символов 
- ввод email после "@" меньше количества допустимых символов
- ввод email после точки перед доменом больше количества допустимых символов 
- ввод email после точки перед доменом меньше количества допустимых символов
- ввод email с пропуском обязательного пункта хотя бы одной точки
- ввод email с пропуском обязательного пункта - наличие "@"
- ввод email с двумя "@"
- проверка уведомления о уже зарегистрированном пользователе с этим email
- вставить скопированный текст
- перенос строки с помощью tab или enter
- не вводить домен


5.Проверка поля "пароль":
- ввести валидный пароль
- пароль скрыт "*"
- проверка появления подсказки, при вводе не валидного пароля 
- ввод пробел в начале строки
- ввод пробел в середине строки
- ввод пробел в конце строки
- ввод только пробел
- вставить скопированный текст
- перенос строки с помщью tab и enter
- ввод пароль больше количества допустимых символов 
- ввод пароль меньше количества допустимых символов
- ввод пароль с пропуском обязательного пункта хотя бы одной заглавной буквы
- ввод пароль с пропуском обязательного пункта хотя бы одной цифры
- ввод пароль с пропуском обязательного пункта хотя бы одного спец.символа
- ввод не на латинице
- ввод только спецсимволы
- ввод только цифры
- ввод пароля с несколькими спецсимволами подряд
- ввод легкого пароля на проверку наличия инлексатора сложности 
- перенос строки с помощью tab или enter
